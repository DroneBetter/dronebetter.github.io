given a field \(F\), \(F[X]\) denotes the ring of polynomials whose coefficients are in \(F\) (with \(f\in F[X]\) given by \(a_0+\ldots+a_nx^n\) with \(n=\deg(f)\) and \(a_n\ne0\))
we can do truncating big-endian division (ie. the poly.py's moddiv operation), so we can get \(\gcd\)s via Euclid's algorithm (ie. it's a Euclidean domain); we have divisorship and irreducibility as per ordinary ring theory
see the set of commutative ring classes; since Euclidean domains are UFDs, every irreducible polynomial is prime and we have unique factorisation into primes
the is where all prime polynomials are linear (so all polynomials can be factorised maximally); due to the fundamental theorem of algebra, this is always a subset of \(\mathbb C\) (for a real/complex-valued original field)
(prop. 1.2) for a ring \(R\), we have that if \(\alpha=\frac rs\in\mathrm{field_{of}fractions}(R)\) is a root of \(f\in R[X]\), then \(r|[x^0]p\) and \(s|[x^{\deg(p)}]p\) (providing an easy test for getting rational roots, and more generally Kronecker's method/algorithm).
proof: ahaha trivial
we call a polynomial \(f\) monic if \([x^n]f=1\); in this case, all rational roots are integer
given a field \(F\), we write \(F^\times:=F\setminus\{0\}\)
(prop. 1.3) for \(f,g\in F[X]\) s.t. with \(\alpha\in F,\beta\in F^\times\), \(g(X)=f(u(X):=\beta X+\alpha)\), \(f\) is irreducible over \(F\) iff \(g\) is irreducible
in general, this is the case if \(u\) is some polynomial \(F\to F\)
for a prime \(p\), \(\phi:\overset{\mathbb Z}a\overset\to\mapsto\overset{\mathbb F_p}{a\% p}\) is a ring homomorphism.
the function \(\phi^*:\mathbb Z[X]\to\mathbb F_p[X]\) that enacts \(\phi\) on all the coefficients is also a ring homomorphism
proofs: ahahaha
(prop. 1.4, Gauss's lemma) if \(f\in\mathbb Z[X]\) is reducible over \(\mathbb Q[X]\), it is also reducible in \(\mathbb Z[X]\)
proof: Kronecker's algorithm ahahahaha
(prop. 1.5) if \(f\in\mathbb Z[X]\) and \(g\in\mathbb Q[X]\) are both monic and \(g|f\), then \(g\in\mathbb Z[x]\)
proof: special case of 1.4
(prop. 1.6, Eisenstein's criterion) if all but the highest term of \(f\in\mathbb Z[X]\) are divisible by \(p\) but \(p^2\nmid[x^0]f\), then \(f\) is irreducible over \(\mathbb Q[X]\)
(def. 1.7) cyclotomic polynomials\[\Phi_n(X):=\prod_{k=0\atop\gcd(n,k)=1}^nx-e^{\frac kn\tau i}\]has special case \(\Phi_p(X)={1-x^p\over1-x}\) (and for that matter \(\Phi_{p^n}(X)={1-x^{p^n}\over1-x^{p^{n-1}}}\)
(lem. 1.8) Eisenstein's critter is quite useful, telling us \(\Phi_p(X+1)=\sum_{k=0}^{p-1}\binom p{k+1}X^k\) is irreducible; \(\Phi_p\) is too by prop 1.3
(lem. 1.9) if \(f\) is reducible in \(\mathbb Q[X]\) then it's also reducible in \(\mathbb F_p[X]\) for all \(p\in\mathbb P\) (since we can homomorph \(\mathbb Q\) into \(\mathbb F_p\)s by replacing denominators with multiplicative inverses mod \(p\)); thus, by contrapositive we can show that a polynomial is irreducible in \(\mathbb Q[X]\) by finding a prime \(p\) such that it's irreducible
